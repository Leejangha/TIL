# GIT

- working directory = 내 컴퓨터
- staging area = 임시 공간
- repository = 외부 컴퓨터

- add -> 내 컴퓨터에서 임시 공간으로 추가한다
- commit (추가한 것 에다가 변경 사항 또는 메모를 추가한다) - 쪽지
- push -> 임시 공간 -> 외부 컴퓨터

- git init : 초기화
- git add READEME.md
- git status : 상태 확인
- git commit -m “first commit”
- git remote add origin url : url을 origin에 대입, 외부 공간으로 지정
- git remote -v : 버전 확인
- remote : 원격조작
